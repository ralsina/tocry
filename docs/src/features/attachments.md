# File Attachments

Add files and images to your notes with simple upload functionality.

## Upload Methods

ToCry provides basic ways to add files to your notes:

### Drag & Drop

- **Direct Upload**: Drag files directly from your computer onto any note
- **Multiple Files**: Select and drag multiple files at once
- **Visual Feedback**: See upload progress and status indicators

### Copy & Paste

- **Image Paste**: Paste images directly from clipboard (Ctrl/Cmd + V)
- **Screenshot Paste**: Take screenshots and paste them immediately
- **File Paste**: Copy files from file explorer and paste into notes

## File Support

ToCry treats most files as generic blobs with special handling for images:

### Images

- **Common Formats**: JPEG, PNG, GIF, SVG
- **Display**: Images are displayed inline within notes
- **Resizing**: Resize images with simple controls

### Other Files

- **Generic Upload**: Any file type can be uploaded
- **Download Links**: Non-image files appear as download links
- **No Preview**: No built-in preview functionality for documents or other file types

## Storage & Security

- **Local Storage**: All files stored securely on your server
- **File Size Limits**: Configurable upload size limits
- **Access Control**: File permissions inherit from note/board settings
- **Basic Organization**: Files organized by upload date
