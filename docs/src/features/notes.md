# Notes & Cards

Create rich, detailed tasks with advanced formatting and management capabilities.

## Rich Content Editor

ToCry features a powerful WYSIWYG Markdown editor with live preview and extensive formatting options:

### Text Formatting

- **Markdown Support**: Full Markdown syntax with live preview
- **Syntax Highlighting**: Code blocks with language-specific highlighting
- **Rich Text Editing**: Bold, italic, headings, lists, and more
- **Tables**: Create structured tables with Markdown syntax

### Media Integration

- **Image Uploads**: Drag & drop images directly into notes
- **File Attachments**: Upload any file type as an attachment
- **Paste Images**: Paste images from clipboard directly
- **Image Resizing**: Resize images with simple controls
- **Attachment Preview**: Preview documents and media files

## Note Organization

### Priority Management

- **Priority Labels**: High, Medium, Low priority indicators
- **Visual Priority**: Color-coded priority badges
- **Priority Filtering**: Sort and filter by priority level
- **Priority Templates**: Quick priority assignment based on content type

### Date Management

- **Start/End Dates**: Set planned start and completion dates
- **Smart Display**: Dates only show when note is expanded
- **Date Validation**: Prevent invalid date ranges
- **Calendar Integration**: Visual date picker interface

### Smart Notes

- **Collapsible Notes**: Reduce clutter by collapsing note content
- **Auto-expand**: Notes expand when you start editing
- **Quick Preview**: Hover to see note content without expanding
- **Note Templates**: Pre-configured note formats for common tasks
- **Duplicate Notes**: Quickly create similar notes from existing ones

## Advanced Features

### Version History

- **Automatic Saving**: Notes are saved automatically as you type
- **Change Tracking**: Track all changes to note content
- **Version Comparison**: See differences between versions
- **Restore Points**: Restore previous versions when needed

### Search & Discovery

- **Content Search**: Find text within note content
- **Tag Search**: Search by custom tags and labels


## Collaboration Features

### Note Sharing

- **Link Sharing**: Generate shareable links to individual notes
- **Board Sharing**: Share entire boards with team members
- **Public Boards**: Make boards publicly accessible

### Real-time Updates

- **Live Collaboration**: See changes in real-time when multiple users are editing
- **Conflict Resolution**: Automatic handling of simultaneous edits
