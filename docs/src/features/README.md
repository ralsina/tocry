# Features Overview

ToCry is a powerful, self-hosted Kanban board application designed for modern teams and personal productivity.

## üöÄ Core Features

### **[Kanban Boards](boards.md)**
- **Multi-Board Support**: Create unlimited boards for different projects
- **Drag & Drop Interface**: Natural workflow with visual task management
- **Custom Lanes**: Organize work with customizable workflow stages
- **Board Analytics**: Track throughput, cycle time, and bottlenecks

### **[Notes & Cards](notes.md)**
- **Rich Content Editor**: WYSIWYG Markdown with live preview
- **Priority Management**: High/Medium/Low priority with visual indicators
- **Smart Dates**: Start/end dates that show only when needed
- **Collapsible Notes**: Reduce clutter with intelligent note display

### **[File Attachments](attachments.md)**
- **Drag & Drop Uploads**: Add files and images effortlessly
- **Multiple File Types**: Support for documents, media, and code files
- **Image Paste**: Paste images directly from clipboard
- **Preview & Playback**: Inline previews for documents and media

### **[Real-time Updates](realtime.md)**
- **Live Collaboration**: Instant updates across all connected devices
- **User Presence**: See who's currently viewing or editing
- **Conflict Resolution**: Automatic handling of simultaneous edits
- **Activity Feeds**: Timeline of all recent changes

### **[Public Boards](public-boards.md)**
- **Sharing Options**: Public, read-only, or password-protected access
- **Custom Branding**: Match your organization's visual identity
- **Embed Support**: Integrate boards in external websites
- **Access Controls**: Granular permissions and security options
